# GearGuard

GearGuard is a maintenance management system designed to handle corrective and preventive maintenance in an industrial setup. It supports role-based workflows for employees, technicians, managers, and admins.

The system allows employees to raise maintenance issues, technicians to resolve assigned work, and managers/admins to plan and manage maintenance operations.

---

## Tech Stack

- MongoDB
- Express.js
- React (Vite)
- Node.js

---

## Core Features

- Role-based authentication (Admin, Manager, Technician, Employee)
- Maintenance request creation (corrective and preventive)
- Kanban board for maintenance workflow
- Preventive maintenance calendar
- Equipment and work center association
- Backend-enforced permission rules
